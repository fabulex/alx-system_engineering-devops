# 0x03 Shell Variables and Expansions

## Project Overview
This project focuses on learning **shell variables and expansions** as part of the ALX System Engineering & DevOps curriculum. It contains practical exercises that demonstrate the use of shell scripting to manipulate variables and create simple commands.

---

## Files in this Directory

- **0-alias**:
  A shell script that creates an alias `ls` that executes `rm *`. (For learning purposes; be careful!)

- **1-hello_you**:
  A shell script that prints "hello" followed by the current Linux user.
